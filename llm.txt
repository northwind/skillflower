# Skillflower OpenClaw Skill

Skillflower is a public OpenClaw skill that lets an agent help users order
florist-backed flowers through a hosted Skillflower service.

## Install Surface

- Main skill file: `SKILL.md`
- Manual install target: `~/.openclaw/workspace/skills/skillflower/SKILL.md`
- Required runtime command: `curl`
- Default service URL: `https://skillflower-public-180536915393.us-central1.run.app`
- Optional service override: `SKILLFLOWER_BASE_URL`
- Required API key before ordering:
  - `SKILLFLOWER_API_KEY`, or
  - cached key at `~/.skillflower/api-key`

The API key cannot be generated locally by the skill. It must be issued or
accepted by the hosted Skillflower service.

## User Capability

The user can ask an agent to send flowers for an occasion. The skill collects
recipient details, recommends a bouquet, creates checkout, returns a browser
payment link, and checks order status.

## Ordering Path

1. Create sender.
2. Create receiver and US or Canada delivery address.
3. Create occasion with date, recurrence, tone, and budget.
4. Recommend bouquet.
5. Create checkout session.
6. Return browser checkout URL or confirm with tokenized payment.
7. Look up order status.

## Safety Rules

- Browser checkout is the default public payment path.
- Never collect raw card number, CVV, or bank data in chat.
- Never fabricate products, checkout sessions, provider order ids, payment success, or order status.
- If Skillflower returns an API error, report it exactly and stop.
