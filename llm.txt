# Skillflower OpenClaw Skill

## Purpose

Skillflower lets OpenClaw agents help users order florist-backed flowers through
a hosted Skillflower service.

## User-Facing Capability

Users can ask an agent to send flowers for an occasion. The skill helps the
agent collect delivery details, recommend a bouquet, create checkout, return a
browser payment link or confirm a tokenized payment, and check order status.

## Required Runtime

- `SKILLFLOWER_BASE_URL`
- `SKILLFLOWER_API_KEY`
- `curl`

## Boundaries

- Do not ask for raw card number, CVV, or bank data in chat.
- Do not fabricate products, checkout sessions, provider order ids, or order success.
- If the API returns an error, report the exact error and stop.
- If payment cannot be completed inside the agent runtime, return the checkout URL.

## Ordering Flow

1. Create sender profile.
2. Create receiver and US or Canada delivery address.
3. Create occasion with date, tone, recurrence, and budget.
4. Recommend bouquet and select the primary option unless the user chooses another.
5. Create checkout session.
6. Return browser checkout URL or confirm with tokenized payment.
7. Look up and report order status.
